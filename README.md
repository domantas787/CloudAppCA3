# CloudAppCA3

TO START THE APPLICATIONS
INSECURE VERSION
1. download all code and keep it all one IDE.
2. open terminal in IDE and use "node insecure.js" to start the insecure application
3. This application will be found at http://localhost:3000

SECURE VERSION
1. download all code and keep it all in one IDE
2. open terminal in IDE and use "node secure.js" to start the insecure application
3. This application will be found at http://localhost:4000


NOTE: when starting server if "node secure.js" or "node insecure.js" doesnt work you cant try using
"npm run start:insecure" or "npm run start:secure"
